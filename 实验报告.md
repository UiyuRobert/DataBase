### 任务一

- `Student` 表

  - 八位纯数字约束：

    ```sql
    DELIMITER $
    CREATE TRIGGER check_sid_isnum 
    BEFORE INSERT ON Student 
    FOR EACH ROW 
    BEGIN
        IF NEW.Sid NOT REGEXP '^[0-9]{8}$' THEN
            SIGNAL SQLSTATE '45000' 
            SET MESSAGE_TEXT = 'Sid must be 8 digits';
        END IF;
    END$
    DELIMITER ;
    ```
